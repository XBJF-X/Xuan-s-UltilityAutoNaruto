# XUAN（Xuan's UltilityAutoNaruto） 
### 火影忍者自动化日常工具
## 项目原理
基于图像检测识别信息+调度器自动调度任务
## 特点
- 支持主流模拟器分辨率
  - 2560x1440
  - 1920x1080
  - 1600x900
  - 1280x720
- 支持键位自定义
